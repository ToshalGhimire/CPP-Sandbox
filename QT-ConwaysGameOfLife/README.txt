An Implemented GUI of Conways Game of life Using QT Creator 

Used QGraphicsView and QGraphicsRectItem along basic Widgets and concepts of signals and slots to create this program

Program Options include:
	Play: Auto play simumation based on user input speed
	Step: Take one step in the simulation
	Stop: Stop the Simulation
	Restart: restart the simulation

	Extra:
	Roll Dice: Rolls dice and display on the simulation
	Spawn Zombie: spawns a moveable block on the board
