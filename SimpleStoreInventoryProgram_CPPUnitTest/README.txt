

program allows a user electronically shop at a store based on store data givin to program

CSV textfile formated like:
=====// File Format \\ =====

Store Name

Inventory:
Type of item, item name, cost, quantity
Type of item, item name, cost, quantity
Type of item, item name, cost, quantity
etc..

=====\\ End File format // =====




The user then has the choice to do any of the following things:
a) Display inventory
b) Add item to cart
c) Remove item from cart
d) Checkout

Add item is based off its quantity

checkout determined by cash value that user has vs the total cost of items in cart
